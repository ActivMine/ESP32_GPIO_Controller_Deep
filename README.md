# ESP32 GPIO Controller

Универсальный настраиваемый контроллер ввода-вывода на ESP32.

## Возможности

- Динамическая конфигурация GPIO через веб-интерфейс
- Два режима работы: вход (с подавлением дребезга) и выход (с возможностью сохранения состояния)
- Управление по WiFi (точка доступа или подключение к роутеру)
- Веб-интерфейс Single Page Application (SPA)
- Хранение конфигурации в NVS (Non-Volatile Storage)

## Требования

### Оборудование
- Плата ESP32 (например, ESP32-WROOM-32)
- Компьютер с USB-портом для программирования

### Программное обеспечение
- PlatformIO
- Библиотеки (устанавливаются автоматически в PlatformIO):
  - ArduinoJson (v7+)
  - WebSockets
  - LittleFS
